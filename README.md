# node-dev-template
## Install Nodemon
```
$ npm install -g nodemon
```

## Setup
```
$ cd desktop
$ git clone git@github.com:tyoshikawa1106/node-dev-template.git
$ cd node-dev-template
$ npm i
$ gulp setup
```

## Run
```
$ nodemon
$ gulp
```

## Access
<a href="http://localhost:8080/">http://localhost:8080/</a>
